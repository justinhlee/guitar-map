This is an interactive application to view finger positions of chord progressions on a guitar fretboard.

Currently, as a rough prototype, it supports major, m, maj7, and min7 chords.

Text input format should adhere to standard chord notation. The sharp symbol and lowercase b are used to denote accidentals.

Some examples:

  • Bbmin7
  • C#maj7
  • C
  • Am
